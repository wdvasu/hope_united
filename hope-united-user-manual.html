Hope United

Activity Tracking System - User Manual

Overview

The Hope United Activity Tracking System helps you record and report on daily activities and attendee participation. The system supports both individual check-ins and group event tracking, with comprehensive reporting capabilities.

Quick Reference: Need to add bulk attendees? Use "Group Event Entry" in the admin dashboard. Perfect for outside training events, community gatherings, or any time you have a headcount but no individual names. All bulk entries automatically count toward daily, monthly, and yearly totals.

System Access

Main Navigation

When you open the application at http://[your-server]:3000, you'll see the home page with two main options displayed as cards:

  • Registration - Start a new participant registration
  • Daily Activity - Record a daily activity entry (check in for activities)

Access the admin dashboard at /admin for reporting and management tools (staff only).

For administrative functions, navigate directly to /admin in your browser.

Registration

How to Register a New Participant

  1. From the home page, click "Registration"
  2. Scan the participant's QR code or enter their information manually
  3. Complete all required fields including:
    • Full name and ZIP code
    • Demographic information (veteran status, sexual orientation, gender, race, ethnicity)
    • County (Summit, Stark, Portage, Cuyahoga, Medina, Other OH County, Out of State, or Refused)
    • Optional birth year
    • Waiver agreement
  4. Review and submit the registration
  5. The participant is now in the system and can log activities
Note: Registration is a one-time process. Once registered, participants can check in for activities using their QR code or name.

Activity Tracking

Individual Check-In

For participants checking themselves in:

  1. Click "Open Activity Tracking" from the home page
  2. Scan your QR code or enter your name to log in
  3. Select one or more activity categories that apply to your visit
  4. Click "Save Selected Activities"
Note: Individual check-in is for single participants only and automatically counts as 1 person.

Manual Activity Entry (Admin)

For staff recording activities for specific registered participants:

  1. Navigate to /admin and click "Manual Activity Entry"
  2. Select the date of the activity
  3. Search for and select the participant by name
  4. Choose one or more activity categories
  5. Click "Save Activities"
Note: Manual Activity Entry is for recording individual registered participants only (counts as 1 person). For group events or bulk entries, use "Group Event Entry" instead.

Group Event Entry (Admin)

For staff recording large group events where you don't have individual participant names:

  1. Navigate to /admin and click "Group Event Entry"
  2. Select the date of the event
  3. Choose one or more activity categories (e.g., "Social Event", "Training/Focus Group", "Community Service")
  4. Enter the total number of attendees - This is the total count for the anonymous group (e.g., 50 people at a community picnic)
  5. Click "Save Group Event"

Viewing and Deleting Group Events

The Group Event Entry page shows a "Recent Group Events" section displaying the last 50 entries:

  • View date/time, category, and attendee count for each entry
  • Click the red "Delete" button to remove any erroneous entry
  • A confirmation dialog will appear before deletion
  • The list refreshes automatically after deletion or when new events are saved

This form is specifically designed for recording activities without linking to registered participants - perfect for large events, community gatherings, outside training events, or drop-in activities where you only have a headcount.

Use Case - Outside Training Events: If you trained 50 people at an outside community event, use Group Event Entry to record them in bulk. These 50 people will automatically be added to the total people count for that day, which rolls up into the monthly and yearly totals. No need to register them individually.

Administrative Functions

Admin Dashboard

Access the admin dashboard at /admin. Available functions:

  • Registrations - View all registered participants
  • Activity Report - View monthly activity summaries by category
  • Activity by Person (Date Range) - Generate detailed reports for specific participants or date ranges
  • Manual Activity Entry - Record activities for specific registered participants
  • Group Event Entry - Record anonymous group activities without participant names
  • Open Demo Kiosk - Launch a demo session for testing

Reports

Activity Report

The main activity report shows:

  • Monthly totals for each activity category
  • Grand totals by category and month
  • Year-over-year comparisons (switch years using the year picker)
  • Detailed daily breakdowns (expand each month to see day-by-day counts)
Note: All totals reflect the actual number of people who attended based on the attendee count entered for each activity.

Activity by Person Report

This report provides detailed information about individual participants:

  • Filter by date range, name, ZIP code, or demographics
  • View total activities per person
  • See activity breakdown by category
  • View detailed activity history with timestamps
  • Export data for further analysis

The report includes:

  • Total People - Number of unique participants
  • Total Visits - Total attendance count (sum of all attendee counts)
  • Total Unique Person Visits - Number of unique person-day combinations

Excel Export

Click "Download XLSX" on the Activity Report page to export a spreadsheet with:

  • All activity categories
  • Monthly counts
  • Grand totals
  • Formatted for easy viewing and analysis in Excel

Best Practices

For Accurate Reporting

  1. Always enter the correct number of attendees when recording group activities. This ensures accurate people counts in reports.
  2. Record activities on the day they occur when possible
  3. Use the manual entry feature for backdating activities if needed
  4. Select all applicable categories for each visit
  5. Review reports regularly to ensure data accuracy

For Group Events

  • Count all attendees before entering the activity
  • Choose the most relevant category (e.g., "Social Event" for community gatherings, "Training/Focus Group" for outside training events)
  • Use "Group Event Entry" for bulk recording - one entry can record any number of people
  • Enter the total count in the "Number of Attendees" field
  • All bulk entries automatically roll up into daily, monthly, and yearly totals
  • You can optionally record individual named participants and still add anonymous group counts

For Privacy and Data Security

  • Always log out of the admin dashboard when finished
  • Never share login credentials
  • Respect participant privacy when viewing reports
  • Use anonymous group counts when individual names are not needed

Frequently Asked Questions

Q: What's the difference between Total Visits and Total Unique Person Visits?

A: Total Visits counts the sum of all attendee counts entered (so a group event with 50 people counts as 50 visits). Total Unique Person Visits counts unique combinations of person and day (so if someone attends multiple activities on the same day, it only counts once).

Q: How do I record an event where I don't have names for all attendees?

A: Use the "Group Event Entry" form in the admin dashboard. This form is specifically designed for recording anonymous group activities - just enter the total attendee count and select the categories. No participant names required.

Q: We trained 50 people at an outside event. How do I add them in bulk so they count toward our monthly totals?

A: Use "Group Event Entry" in the admin dashboard. Select the date, choose "Training/Focus Group" (or relevant category), enter "50" as the attendee count, and save. These 50 people will automatically be included in your daily total for that date, which rolls up into your monthly and yearly totals. You don't need to register them individually or enter 50 separate activities.

Q: Can I edit or delete activities after they're entered?

A: Currently, activities cannot be edited or deleted through the web interface. Contact your system administrator if you need to make corrections.

Q: What happens if I forget to log someone's activity?

A: Use the Manual Activity Entry function in the admin dashboard. Select the correct date, find the participant, choose the categories, and save.

Q: How far back can I view activity data?

A: The system stores all historical data. Use the year picker in the Activity Report to view different years, or use date range filters in the Activity by Person report.
